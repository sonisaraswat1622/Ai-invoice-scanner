AI Invoice & Receipt Scanner
An AI-powered computer vision tool that eliminates manual invoice data entry
Built with the Claude Vision API, this browser-based app reads any invoice or receipt image and instantly extracts structured financial data, exporting it directly to Excel or CSV in seconds.
Problem It Solves
Finance teams, accountants, small business owners, and operations managers spend enormous time manually typing invoice data into spreadsheets. This tool automates that entire process using AI vision, saving hours of work per week.

Features
1.	Upload any invoice or receipt photo (JPG, PNG)
2.	AI extracts vendor, date, invoice number, line items, quantities, prices, tax & total
3.	One-click export to Excel (xlsx) with Summary and Line Items sheets
4.	One-click CSV export for any system integration
5.	Runs entirely in the browser — no backend, no installation, no login
